# Railway Semaphore - MicroPython Stage 1

This project simulates a single railway semaphore using MicroPython with three LEDs (Red, Yellow, Green).

## Features
- Single semaphore simulation
- Control each LED individually
- Ready for future expansions

## Project Structure
- `src/main.py` : main program
- `src/config.py` : LED pins and timings
- `docs/` : diagrams

## How to Run
1. Flash MicroPython firmware on ESP32 or Raspberry Pi Pico
2. Copy files from `src/` to device
3. Run `main.py`
